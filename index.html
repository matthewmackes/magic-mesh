Magic Mesh dnf repository

Secure no-fixed-center workgroup mesh on Fedora-Cosmic. GPL-3.0-or-later.

The main magic-mesh RPM installs this repo + key automatically. To add it manually on Fedora:

sudo curl -o /etc/yum.repos.d/magic-mesh.repo \
  https://matthewmackes.github.io/magic-mesh/magic-mesh.repo
sudo dnf install magic-mesh

Or one-shot from the release asset: sudo dnf install https://github.com/matthewmackes/magic-mesh/releases/download/magic-mesh-v10.0.0/magic-mesh-10.0.0-1.x86_64.rpm
